### Функционал:
- система регистрации и аутентификации;
- редактирование профиля;
- списки «Смотрю», «Запланировано» и «Просмотрено»;
- оценка и комментирование контента;
- поиск фильмов по названию, жанрам, актёрам, режиссёрам.
